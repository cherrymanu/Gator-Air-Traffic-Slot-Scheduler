## How to Add Images to Your Report:

### Step 1: Download Images
1. Search using the terms above
2. Download high-quality PNG or JPG images
3. Save them in your project folder: `/Users/surendrakalluri/Cherry/ADS_project/`

### Step 2: Add to Markdown
Uncomment the placeholder lines in `Report.md`:

```markdown
![Description](image_filename.png)
```

### Step 3: Convert to PDF
When you convert `Report.md` to PDF, the images will be embedded automatically.

## Quick Links to Good Sources:

### Educational Sites with Free Diagrams:
1. **GeeksforGeeks** - https://www.geeksforgeeks.org/
   - Excellent data structure diagrams
   - Free to use for educational purposes

2. **Programiz** - https://www.programiz.com/dsa
   - Clean, simple diagrams
   - Good for heap visualizations

3. **VisuAlgo** - https://visualgo.net/en
   - Interactive animations
   - Can take screenshots

4. **Wikipedia** - https://en.wikipedia.org/
   - Usually has diagrams in Commons
   - Check license (usually CC-BY-SA)

5. **TutorialsPoint** - https://www.tutorialspoint.com/
   - Simple, clear diagrams

### Create Your Own (Recommended for Best Fit):
1. **Draw.io** - https://app.diagrams.net/
   - Free online diagram tool
   - Export as PNG/PDF
   - Perfect for architecture diagrams

2. **Excalidraw** - https://excalidraw.com/
   - Hand-drawn style diagrams
   - Quick and easy

3. **PlantUML** - https://plantuml.com/
   - Text-to-diagram tool
   - Great for state machines

## Tips:

1. **Image Quality:** Use high-resolution images (at least 1000px wide)
2. **Relevance:** Make sure the diagram matches your implementation
3. **Attribution:** If required by license, add a note in the caption
4. **File Size:** Keep images under 1MB each for PDF size
5. **Format:** PNG is best for diagrams (better than JPG for text/lines)
